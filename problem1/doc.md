# Problema 1: O Jantar dos Filósofos
> Implementar o problema do Jantar dos Filósofos com N filósofos, utilizando channels para representar os
garfos.

### Requisitos:
- [x] Implementar uma versão base que demonstre a ocorrência de deadlock.
- [ ] Implementar pelo menos 2 estratégias distintas que eliminem o deadlock.
- [ ] Para cada versão (base + estratégias), apresentar uma análise de deadlock e starvation: identificar se o problema ocorre ou não e justificar com base nos conceitos vistos em aula.
- [ ] Coletar dados de execução (ex.: número de vezes que cada filósofo comeu ao longo de N iterações) e usar esses dados para argumentar sobre a fairness de cada estratégia.
